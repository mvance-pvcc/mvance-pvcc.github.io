Operating Systems Terminology
Operating System The software that supports a computer's basic functions, such as scheduling tasks, executing applications, and controlling peripherals.
Job Queue A list of tasks to be carried out in turn by an operating system.
FIFO First-in, first-out (FIFO) is a valuation method in which the assets produced or acquired first are sold, used, or disposed of first.
Interactive Processing In a computer system, the process of interacting directly with a program and/or data for immediately modifying, retrieving and/or displaying information. This is usually done on a computer screen.
Time-Sharing Multiprocessing Time Sharing is the logical extension of multiprogramming, in this time sharing Operating system many users/processes are allocated with computer resources in respective time slots.
Application Software Application software consists of the programs for performing tasks particular to the machine’s utilization.
System Software System software performs those tasks that are common to computer systems in general.
Utility Software Utility software consists of sofware units that extend the capabilities of the operating system.
Graphical User Interface (GUI) These systems allow users to issue commands by using one of several common input devices.
Shell Older user interfaces, called shells, communicated with users through textual messages using a keyboard and monitor screen.
Kernel An operating system's kernel contains those software components that perform the very basic functions required by the computer installation
Device Driver Software units that communicate with the controllers to carry out operations on the peripheral devices attached to the machine.
File Manager Software units that communicate with the controllers (or at times, directly with peripheral devices) to carry out operations on the peripheral devices attached to the machine.
Memory Manager Still another component of an operating system’s kernel is the memory manager, which is charged with the task of coordinating the machine’s use of main memory.
Virtual Memory Still another component of an operating system’s kernel is the memory manager, which is charged with the task of coordinating the machine’s use of main memory.
Boot Strapping Bootstrapping is the technique for producing a self-compiling compiler – that is, a compiler (or assembler) written in the source programming language that it intends to compile.
ROM Read-only memory, or ROM, is a type of computer storage containing non-volatile, permanent data that, normally, can only be read, not written to.
Deadlock When two or more processes are blocked from progressing because each is waiting for a resource that is allocated to another.
Spooling To spool (which stands for "simultaneous peripheral operations online") a computer document or task list (or "job") is to read it in and store it, usually on a hard disk or larger storage medium so that it can be printed or otherwise processed at a more convenient time.
Super User (or Administrator) This person gains highly privileged access to the operating system by identifying him- or herself as the administrator (usually by name and password) during the login procedure. Once this contact is established, the administrator can alter settings within the operating system, modify critical software packages, adjust the privileges granted to other users, and perform a variety of other maintenance activities that are denied normal users.
Multi-Factor Authentication An electronic authentication method in which a user is granted access to a website or application only after successfully presenting two or more pieces of evidence to an authentication mechanism: knowledge, possession, and inherence.

Networking & The Internet
Networking Classifications LAN LAN normally consists of a collection of computers in a single building or building complex.
MAN A MAN is a network of intermediate size, such as one spanning a local community.
WAN WAN links machines over a greater distance—perhaps in neighboring cities or on opposite sides of the world.
Network Models Bus Network Machines communicating directly with each other.
Star Network Machines communicating through an intermediary.
Protocols CSMA/CD Ethernet This protocol dictates that each message be broadcast to all the machines on the bus.
CSMA/CA WIFI Protocols defined in IEEE 802.11
Combining Networks Repeater A device that passes signals back and forth between the two original buses (usually with some form of amplification) without considering the meaning of the signals.
Bridge Similar to, but more complex than, a repeater. Like a repeater, it connects two buses, but it does not necessarily pass all messages across the connection. Instead, it looks at the destination address that accompanies each message and forwards a message across the connection only when that message is destined for a computer on the other side.
Switch A switch is essentially a bridge with multiple connections, allowing it to connect several buses rather than just two. Thus, a switch produces a network consisting of several buses extending from the switch, as spokes on a wheel.
Router The connection between networks to form an internet is handled by devices known as routers, which are special purpose computers used for forwarding messages.
Gateway The “point” at which one network is linked to an internet is often called a gateway because it serves as a passageway between the network and the outside world.
Client/Server Model Peer-to-Peer (P2P) The peer-to-peer model involves processes that provide service to and receive service from each other.
Distributed System Consist of software units that execute as processes on different computers.
Cloud Computing Whereby huge pools of shared computers on the network can be allocated for use by clients as needed, is the latest trend in distributed systems.
Internet Addressing Tier 1 Very high-speed, high-capacity, international WANs.
Tier 2 Tier-2 ISPs tend to be more regional in scope and less potent in their capabilities.
Tier 3 Essentially an independent Internet, sometimes called an intranet, operated by a single authority that is in the business of supplying Internet access to individual homes and businesses.
IP Addresses ICANN Blocks of consecutively numbered IP addresses are awarded to ISPs by the Internet Corporation for Assigned Names and Numbers (ICANN), which is a nonprofit corporation established to coordinate the Internet's operation.
DNS If a human wants to send a message to a distant machine and identifies the destination by means of a mnemonic address, the software being used must be able to convert that address into an IP address before transmitting the message. This conversion is performed with the aid of numerous servers, called name servers, that are essentially directories that provide address translation services to clients.
Top-Level Domain (TLD) As an example, the domain name of Marquette University is mu.edu. Note the suffix following the period. It is used to reflect the domain’s classification, which in this case is “educational,” as indicated by the .edu suffix. These suffixes are called top-level domains.
Country Code TLD In the earliest days of Internet TLDs, there was a default assumption that connected organizations resided in the United States. The Internet quickly grew to include countries all over the world, resulting in the creation of new two-letter TLDs for each specific country.
Internet Protocols FTP An application for listing and copying files across a network implements the file transfer protocol.
HTTP Hypertext documents are normally transferred between browsers and webservers using a protocol known as the Hypertext Transfer Protocol.
SMTP A way that two computers on the network may interact when transmitting an email message from one host to the other.
World Wide Web URL In order to locate and retrieve documents on the Web, each document is given a unique address called a Uniform Resource Locator (URL)
Client-Side Activities Activities if they are performed by a client (such as a browser).
Server-Side Activities Activities if they are performed by a server (such as a webserver).
Internet Four-Layer Software Application Layer The application layer consists of those software units such as clients and servers that use Internet communication to carry out their tasks.
Transport Layer The transport layer adds sequence numbers to the small segments it produces so that the segments can be reassembled at the message’s destination. Then it hands these segments, known as packets, to the network layer.
Network Layer It is the network layer’s job to decide in which direction a packet should be sent at each step along the packet’s path through the Internet.
Link Layer The link layer has the responsibility of transferring the packet. Thus, the link layer must deal with the communication details particular to the individual network in which the computer resides.
Internet is: Packet-Switching Network A packet-switched system through which digital data is sent by breaking the data into blocks of bits called packets, which contain both the data being transmitted and control information for routing the data.
Fault Tolerant Routing on the Internet is fault tolerant and redundant.
Redundant The redundancy of routing (i.e., more than one way to route data) between two points on the Internet increases the reliability of the Internet and helps it scale to more devices and more people.
Cybersecurity Malware Examples of software that is transferred to, and executed on, the computer under attack include viruses, worms, Trojan horses, and spyware, whose names reflect the primary characteristic of the software.
Virus A virus is software that infects a computer by inserting itself into programs that already reside in the machine. Then, when the “host” program is executed, the virus is also executed.
Worm A worm is an autonomous program that transfers itself through a network, taking up residence in computers and forwarding copies of itself to other computers.
Trojan Horse A Trojan horse is a program that enters a computer system disguised as a desirable program, such as a game or useful utility package, that is willingly imported by the victim. Once in the computer, however, the Trojan horse performs additional activities that might have harmful effects.
Spyware Software that collects information about activities at the computer on which it resides and reports that information back to the instigator of the attack.
Phishing A technique of obtaining information explicitly by simply asking for it.
DDoS Attack Distributed denial of service (DDoS) attacks are a subclass of denial of service (DoS) attacks. A DDoS attack involves multiple connected online devices, collectively known as a botnet, which are used to overwhelm a target website with fake traffic.
Protection Public Key Encryption An encryption method that is widely used because of the enhanced security associated with its use.
Firewall A primary prevention technique is to filter traffic passing through a point in the network
Spam Filter Firewalls designed to block unwanted email.
Proxy Server A software unit that acts as an intermediary between a client and a server with the goal of shielding the client from adverse actions of the server.
Antivirus Software Software used to detect and remove the presence of known viruses and other infections.
Cryptography The study of sending and receiving messages securely in the presence of adversaries.
